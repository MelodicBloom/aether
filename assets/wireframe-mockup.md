# AETHER Wireframe & Visual Mockup

Generated during Phase 1–2 design planning. Three zones:

## Zone 1 — Gallery Wireframe (Grayscale)

- Top nav: logo, family tabs, search
- Family tabs: Thin-Film | Fresnel | Bubble | Shell | Holographic
- Responsive card grid with free + locked states
- Right-side detail drawer: preview, uniform controls, CTA

## Zone 2 — Docs + Pricing Wireframe (Grayscale)

- Pricing: tier cards ($0 / $79 / $249), feature comparison
- Docs: README sections, code blocks, package contents panel

## Zone 3 — Visual Mockup (High Fidelity)

- Deep charcoal background
- Luminous iridescent shader cards (abalone/nacre shimmer from `shader-gallery/abalone`)
- Soft glass panel overlays with backdrop blur
- Cyan → violet → magenta specular language
- Satoshi + mono type pairing
- Family tabs with active underline
- Detail drawer with live uniform sliders
